# assets

Сюда можно положить свои файлы, чтобы заменить оформление картинки результата
(всё опционально — без них рисуется встроенный градиентный шаблон):

- `background.png` — фон 1080×1350 px
- `font.ttf` — обычный шрифт с поддержкой кириллицы
- `font-bold.ttf` — жирный шрифт

Если файлов нет, бот использует системный шрифт (Arial на Windows / DejaVu на Linux)
и нарисованный программно фон. Дизайн при этом остаётся одинаковым для всех результатов.
